# Research Pipeline MVP Gate Reports

This directory stores MVP gate evaluation reports for the research pipeline.

Reports are generated by `app/research_pipeline/evaluation/run_mvp_gate.py`.

## Report Types

- **JSON reports**: Machine-readable gate evaluation results
- **Markdown reports**: Human-readable gate evaluation summaries

## Usage

```bash
python -m app.research_pipeline.evaluation.run_mvp_gate \
  --db-path app/storage/metadata/research_pipeline.sqlite3 \
  --run-ids run_20240101_120000_000000 run_20240101_130000_000000 \
  --seed-dataset tests/fixtures/seed_dataset.jsonl \
  --output-json app/evaluation/reports/gate_report.json \
  --output-md app/evaluation/reports/gate_report.md
```

## MVP Gate Conditions

From PRD section 10.3:

1. **Completion**: ≥2/3 seed questions reach "completed" status
2. **Time to report**: Median time < 300 seconds (5 minutes)
3. **Reader paper count**: Median ≥ 3 papers read per run
4. **Claim verification coverage**: Mean ≥ 60% of claims verified
