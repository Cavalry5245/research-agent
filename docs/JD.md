# 大语言模型与 Agent 应用开发实习生

岗位职责

1. 协助进行大语言模型相关应用的开发与测试，包括 Prompt 编写、调用链路搭建、效果验证与优化。
2. 参与 Agent 系统的设计与实现，包括任务拆解、工具调用、记忆管理、工作流编排、多轮交互等能力开发。
3. 配合业务场景完成基础数据处理与分析工作，包括数据清洗、统计分析、结果可视化与问题定位。
4. 参与模型/Agent效果评估，整理实验数据，分析失败案例，支持优化迭代。
5. 协助构建知识库、检索增强、工具集成等模块，提升系统可用性与稳定性。
6. 参与接口开发、日志分析、问题排查及文档编写，支持项目工程化落地。
7. 跟踪 LLM / Agent 相关新技术，协助进行方案调研与原型验证。

任职要求

1. 本科及以上学历在读，计算机、软件工程、人工智能、数据科学、自动化、数学等相关专业优先。
2. 熟悉 Python，具备基本的工程开发能力，能够独立完成脚本编写、数据处理与简单模块开发。
3. 具备基础数据分析能力，熟悉常用数据处理方法，能够使用 Pandas、NumPy、Matplotlib 等工具进行数据清洗、统计和可视化。
4. 对大语言模型应用有一定了解，了解 Prompt Engineering、RAG、Function Calling、Tool Use 等基本概念。
5. 对 Agent 方向有兴趣或有相关实践，了解任务规划、工具调用、工作流编排、多Agent协作中的至少一部分内容。
6. 具备良好的逻辑分析能力和问题定位能力，能够基于实验结果进行归因和优化。
7. 具备良好的沟通能力和文档能力，能够清晰表达技术方案、实验现象和分析结论。
8. 每周可实习不少于 4 天，实习周期不少于 3 个月。

加分项

1. 有 LLM 应用、Agent、RAG、知识库、信息抽取、问答系统等相关项目经验。
2. 使用过 LangChain、LangGraph、AutoGen、CrewAI、LlamaIndex、OpenAI SDK、vLLM、FastAPI 等相关框架或工具。
3. 熟悉基础机器学习/NLP方法，了解 embedding、rerank、向量检索、评测指标等。
4. 有数据分析、A/B测试、实验设计、日志分析或效果评估经验。
5. 有一定后端开发能力，了解 API 设计、异步任务、数据库、缓存、消息队列等常见组件。
6. 有较强的自驱力和学习能力，能快速理解新场景并推进问题落地。
